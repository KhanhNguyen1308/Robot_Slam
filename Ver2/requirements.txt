# Notes
# -----
# * opencv-python       — use the JetPack-provided build when possible:
#                         pip3 install opencv-python is fine for dev/testing;
#                         on the Jetson itself the pre-built CUDA-enabled wheel
#                         from JetPack is preferred (do NOT pip-install cv2 if
#                         it was already installed by JetPack).
#
# * Jetson.GPIO         — GPIO access on Jetson Nano.
#                         sudo pip3 install Jetson.GPIO
#                         sudo groupadd -f gpio
#                         sudo usermod -aG gpio $USER
#
# * smbus2              — I2C bus access for MPU6050 IMU.
#
# * ultralytics         — YOLOv8/v11 (inference server only, not needed on Nano
#                         if the Nano only runs server_client.py).
#                         Requires torch; install the JetPack-provided torch wheel first:
#                         https://developer.download.nvidia.com/compute/redist/jp/v46/pytorch/
#
# * espeak-ng / aplay   — TTS audio (system packages, not pip):
#                         sudo apt install -y espeak-ng alsa-utils ffmpeg
#
# * gtts                — Optional high-quality online TTS (needs network + ffmpeg).
#
# * pyttsx3             — Optional offline Python TTS fallback.
# =============================================================================
